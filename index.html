COMPÉTENCES

Laravel

Laravel est un framework de développement web complet et facile à utiliser qui permet de créer rapidement des applications Web de qualité. C'est pour ces raisons que ce framework est le meilleur que je puisse utiliser aujoud'hui.

HTML

HTML est un langage de balisage incontournable car c'est le langage de base pour le développement Web, c'est facile à apprendre et c'est un langage ouvert et standardisé.

CSS

Le CSS est langage de style pour tous les développeurs Web, il offre une grande flexibilité et facilite le developpement et la maintenance de sites Web de qualité.

BOOSTRAP

Bootstrap est un framework de développement Web qui permet de créer des sites Web de manière rapide et facile. Il inclut un grand nombre de modèles de base et de composants prédéfinis qui aide à créer une interface utilisateur attrayante et cohérente sans avoir à partir de zéro.

JAVASCRIPT

JavaScript est un langage de programmation populaire utilisé pour créer des fonctionnalités interactives sur les sites Web. Il est souvent utilisé en conjonction avec HTML et CSS pour créer des pages Web dynamiques et réactives, ce qui en fait un choix polyvalent pour les développeurs.

JAVA

Java est un langage de programmation populaire. Il est très polyvalent, ce qui signifie qu'il peut être utilisé pour développer une grande variété d'applications, que ce soit pour le web, les appareils mobiles ou les ordinateurs de bureau
